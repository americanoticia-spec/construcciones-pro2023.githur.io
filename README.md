# Construcciones Pro2023 - Landing Page

Bienvenido al repositorio de la página web de **Construcciones Pro2023**. Esta es una Landing Page moderna, orientada a la conversión y completamente responsiva, diseñada para ofrecer servicios de gestión integral de proyectos, remodelaciones, reparaciones, instalaciones y mantenimiento en Caracas, El Junquito y Miranda.

## 🚀 Características
*   **Diseño Moderno y Limpio**: Interfaz amigable para el usuario con enfoque en la estética corporativa (Amarillo y Blanco).
*   **Totalmente Responsivo**: Se adapta a cualquier dispositivo (móviles, tablets y computadoras).
*   **Optimización SEO**: Metadatos, etiquetas Open Graph para redes sociales, Twitter Cards y etiquetas H1 optimizadas para mejorar el posicionamiento local en Google.
*   **Botón de WhatsApp Flotante**: Para contacto rápido e inmediato.
*   **Animaciones Suaves**: Efectos al pasar el cursor y un menú de navegación tipo "Sticky" que acompaña al usuario al hacer scroll.

## 🛠️ Tecnologías Utilizadas
*   HTML5 Semántico
*   CSS3 (Estilos puros sin frameworks externos, variables CSS, Flexbox y Grid)
*   JavaScript (Interactividad ligera para el menú móvil y scroll)

## 📝 Guía de Edición (Cómo cambiar tus datos)

Para adaptar esta página con tu información real, solo necesitas abrir el archivo `index.html` en cualquier editor de texto o código (como Notepad, VS Code o Sublime Text) y modificar lo siguiente:

### 1. Cambiar Número de WhatsApp
Busca en `index.html` las siguientes líneas:
*   **Sección de Contacto (aprox. línea 205):** Cambia `+58 412 000 0000` por tu número visible y actualiza el enlace `href="https://wa.me/584120000000"`.
*   **Botón flotante (al final del código, aprox. línea 271):** Cambia el número en el enlace `href="https://wa.me/584120000000?text=Hola..."`. Recuerda usar el código de país sin el símbolo `+`.

### 2. Cambiar Correo Electrónico
*   En la sección de Contacto (aprox. línea 211), busca `info@pro2023.com` y reemplázalo por el tuyo en el texto visible y en el enlace `href="mailto:tu-correo@ejemplo.com"`.

### 3. Cambiar Redes Sociales
*   Busca `https://instagram.com/tu_usuario` (en la sección de Portafolio y en el Footer) y reemplázalo por la URL real de tu cuenta.

### 4. Cambiar Imágenes y Logotipo
*   Para cambiar una imagen, busca las etiquetas `<img src="...">`.
*   Reemplaza la URL que está dentro de `src="..."` por el nombre de tu archivo (ejemplo: `src="logo-pro2023.png"`). Asegúrate de que las fotos que agregues estén guardadas en la misma carpeta que el archivo `index.html`.

## 🌐 Cómo visualizar la página
Solo necesitas descargar o clonar los archivos y abrir `index.html` en tu navegador web preferido (Chrome, Safari, Edge, Firefox). No requiere de servidores complejos para funcionar.
